Random Library Book

Every item in the catalog has an equal shot.

A free random book picker for public library catalogs. Pick up something you never would have searched for — no algorithm, no recommendations, no filter bubble. Every physical item in the collection has an equal shot.

Supported libraries:

  • Boston Public Library (Boston, MA) — BiblioCommons catalog
  • Detroit Public Library (Detroit, MI) — SirsiDynix catalog
  • San Francisco Public Library (San Francisco, CA) — BiblioCommons catalog
  • Chicago Public Library (Chicago, IL) — BiblioCommons catalog
  • Seattle Public Library (Seattle, WA) — BiblioCommons catalog
  • Toronto Public Library (Toronto, ON) — BiblioCommons catalog

Also available as a command-line tool. Run python3 main.py --help for usage. Source code on GitHub.
